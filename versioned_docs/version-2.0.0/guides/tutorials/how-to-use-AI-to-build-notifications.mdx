# How to Use AI to build notifications

> Courier's AI content generator (backed by best-in-class GPT-3 models) lets you build and iterate on notification content rapidly.

## Creating new notifications

### Using an intuitive Notification title

The title you use for the new notification would be an input to the AI which will determine the generated content. Enter a notification title and check the Use AI checkbox.

<Image
  img={require("../../assets/guides/courier-app/notification-title.png")}
  alt="Notification title"
/>

Examples of titles you can use:

- Password reset request
- Login verification
- Subscribe to our newsletter
- Forgot password
- Upgrade pricing tier
- Add your social media
- Free tier limit exceeded
- New feature launch

### Generated Notification

Courier AI content generator would build a notification with multiple text blocks and email channel by default. You can attach a provider or switch to a different channel.

<Image
  img={require("../../assets/guides/courier-app/generated-notification.png")}
  alt="Generated Notification"
/>

## Editing notification content

You can use AI to edit notification content using content generator toolbar

### Click on the brain icon on text block toolbar

A text block toolbar includes a brain icon which opens an AI content generator modal.

<Image
  img={require("../../assets/guides/courier-app/content-generator-toolbar.png")}
  alt="Content generator toolbar"
/>

### Insert a prompt and click generate content

Insert a prompt which provides some context to the AI. Then click Generate Content.

<Image
  img={require("../../assets/guides/courier-app/content-generator-toolbar-prompt.png")}
  alt="Content generator toolbar prompt"
/>

### Add content to the notification

Use the generated content as is or feel free to edit it. Then click Add Content to apply it on the text block.

<Image
  img={require("../../assets/guides/courier-app/content-generator-toolbar-output.png")}
  alt="Content generator toolbar output"
/>

Text block content would be replaced. Feel free to use the generator again or update the content directly.

<Image
  img={require("../../assets/guides/courier-app/content-generator-toolbar-output-applied.png")}
  alt="Content generator toolbar output applied"
/>
