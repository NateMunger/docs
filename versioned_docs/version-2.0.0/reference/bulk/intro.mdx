# Bulk API

Bulk processing allows you to send message definitions to large numbers of users in a single API invocation.

New to bulk processing? Check out [this guide](https://www.courier.com/docs/guides/tutorials/how-to-send-bulk-notifications/) to start sending bulk notifications in minutes.
