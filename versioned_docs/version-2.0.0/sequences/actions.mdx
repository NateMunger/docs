# Actions

## Send

Send a message.

### Fields

| Field   | Type   | Required | Description                                                                                    |
| ------- | ------ | -------- | ---------------------------------------------------------------------------------------------- |
| action  | string | required | The type of sequence action. For the Send Action, this value should be `"send"`.               |
| message | object |          | The courier message object. See the [Send Api Spec](../../reference/send/message) for details. |

### Schema

```json
{
  "action": "send",
  "message": {}
}
```
